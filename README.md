# jupyter-gpp-kernel
This is a basic Jupyter kernel for [c++](https://gcc.gnu.org/) and [PlantUML](https://plantuml.com/) notebooks. It can render [SVG](https://en.wikipedia.org/wiki/SVG) and [PNG](https://en.wikipedia.org/wiki/PNG) diagrams.
```
jupyter kernelspec install --user jupyter-gpp-kernel
export PYTHONPATH=~/.local/share/jupyter/kernels:$PYTHONPATH
```
